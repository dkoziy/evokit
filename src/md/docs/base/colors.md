|   КЛЮЧ    | ЗНАЧЕНИЕ |
|-----------|----------|
| `base`    | #3c3c3c  |
| `link`    | #5e8fd5  |
| `red`     | #f55832  |
| `yellow`  | #facd00  |
| `muted`   | #a0b0c7  |
| `info`    | #5e8fd5  |
| `success` | #33b600  |
| `warning` | #ff9600  |
| `error`   | #bb2914  |
