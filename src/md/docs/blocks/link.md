# Link

Блок предназначен для создания ссылок.

---

## Использование

``` html
<div class="doc-example">
    <a class='ui-link'>Cсылка</a>
</div>
```

---

## Стили

|          Class          |             Description             |
|-------------------------|-------------------------------------|
| `.ui-link_style_dotted` | Добавляет пунктирное подчеркивание. |

``` html
<div class="doc-example">
    <a class='ui-link ui-link_style_dotted'>Пунктирная ссылка</a>
</div>
```
