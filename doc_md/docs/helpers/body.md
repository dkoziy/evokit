# Body

Добавьте класс `.ui-body` для тега `<body>`, чтобы определить дефолтные стили на странице.


|         Name        |             Value            |
|---------------------|------------------------------|
|  `font-family`      |  "Roboto", Arial, sans-serif |
|  `font-size`        |  13px                        |
|  `line-height`      |  1.3em                       |
|  `color`            |  #3c3c3c                     |
|  `background-color` |  #fff                        |
|  `min-width`        |  1000px                      |
