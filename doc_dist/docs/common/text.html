Class
Description

.ui-text-underline
Подчеркнутый текст.

Курсивный текст.

Выравнивание текста

Добавьте один из этих классов для выравнивания текста по горизонтале.
Class
Description

Выравнивает текст влево.
.ui-text-center
Выравнивает текст по центру.
.ui-text-right
Выравнивает текст вправо.

Перенос текста

Добавьте один из этих классов, чтобы определить перенос текста.
Class
Description
.ui-text-ellipsis
Текст обрезается и к концу строки добавляется троеточие.
.ui-text-break
Перенос строк чтобы текст поместился в блок.
.ui-text-nowrap
Весь текст отображается одной строкой.
